# Task 2: Timeline & Audio Synchronization

## Core Requirements
- [ ] **Timeline UI:**
    - [ ] Implement a full-width timeline at the bottom of the screen.
    - [ ] Waveform visualization (optional but highly recommended for rhythm syncing).
    - [ ] Scrolling and Zooming functionality (`Ctrl + Scroll` for zoom).
- [ ] **Playback Controls:**
    - [ ] Play/Pause (`Space`).
    - [ ] Scrubbing (drag cursor on timeline).
    - [ ] Seek functionality (`Left/Right` keys for 5s, `Shift + Left/Right` for precise frame steps).
    - [ ] Playback speed adjustment (e.g., 25%, 50%, 75%, 100%).
- [ ] **Marker Visualization:**
    - [ ] Render node markers on the timeline (different colors for Normal vs Slider).
    - [ ] Render Wall markers.
    - [ ] Render Timing Point (BPM change) markers.
- [ ] **Beat Alignment:**
    - [ ] Visual grid based on BPM (1/4, 1/8, 1/16 beats).
    - [ ] "Snap to Grid" functionality for precise node placement.
