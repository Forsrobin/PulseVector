# Task 3: Placement Tools & Event Editing

## Core Requirements
- [ ] **Viewport Management:**
    - [ ] Render the 360-degree gameplay field in the editor.
    - [ ] Scrub the viewport state as the timeline cursor moves.
- [ ] **Placement Tools:**
    - [ ] **Normal Node:** Click to place a node at current time and 3D coordinate (x, y).
    - [ ] **Slider:** Click to place start, then drag/click for curve points.
    - [ ] **Wall:** Hotkey to spawn a collapsing wall event.
- [ ] **Property Inspector:**
    - [ ] Sidebar for editing the currently selected object.
    - [ ] Modify Node type (Normal <-> Slider).
    - [ ] Modify Slider duration and curve.
    - [ ] Modify Wall properties (speed, openness/closeness, duration).
- [ ] **Interaction:**
    - [ ] Drag nodes in the viewport to change position (X/Y).
    - [ ] Multi-select nodes (drag-box or Ctrl + Click).
    - [ ] Delete nodes (`Delete`/`Backspace`).
    - [ ] Copy/Paste nodes (`Ctrl + C`, `Ctrl + V`).
