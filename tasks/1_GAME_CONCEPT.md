# Task 1: High-Level Game Concept Implementation

## Core Requirements
- [ ] Implement a 360-degree spatial field for node spawning.
- [ ] Develop a mouse-based continuous tracking system (aiming/tracing).
- [ ] Implement a keyboard-based discrete temporal execution system (rhythm/frequency matching).
- [ ] Design and implement node frequencies mapped to keys (e.g., A, S, D, F).
- [ ] Create spline-based movement paths for nodes traveling towards/away from the "Event Horizon".
- [ ] Implement "Sliders/Tethers" requiring both sustained key presses and precise mouse tracing.
- [ ] Develop the "Thunk" feedback system:
    - [ ] Hitstop (2-3 frame pause on perfect hit).
    - [ ] Localized radial blur shader trigger.
    - [ ] Vector particle burst emission.
